1.1:Why is one of the symbols used for exclusive disjunction the “not equal to” sign?

One of the symbols is the “not equal to” sign because in exclusive disjunction, only one output can be true, but both outputs can not be true at the same time. Therefore, A cannot be equal to B, hence the “not equal to” sign.

2.1:Why is the formula for determining the number of rows in a truth table related only to the number of inputs, and why is the formula 2 to the power of n(number of inputs)?

The formula for determining the number of rows in a truth table is related only to the number of inputs since each input can have a max of two outputs, which are true and false. Therefore, it is unnecessary to have anything more in the formula.The formula is 2 to the power of n since one input can have two different outputs at any given time.
